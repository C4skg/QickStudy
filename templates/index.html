函数

一、概念

  1. 定义

    自变量 x ，因变量 y ，其中 x 通过一个对应法则与 y 一一对应，则记为 y=f(x)

二、函数定义域

在一个函数 y=f(x) 中 x 的取值范围

常见函数定义域如下

  • f\left(\left[\frac{1+\left\{x,y\right\}}{\left(\frac{x}{y}+\frac{y}{x}\right)\left(u+1\right)}+a\right]^{\frac{3}{2}}\right)
  • y=\sqrt[2n]{x}\Rightarrow(x\geq0)
  • y=\sqrt[2n+1]{x}\quad\Rightarrow (x\in(-\infty,+\infty),x\in R)
  • y=\log_a{x}\Rightarrow(x>0) \quad
  • y=\tan{x}\Rightarrow(x \neq k\pi+\frac{\pi}{2})
  • y=\cot{x}\Rightarrow(x\neq k\pi)
  • y=\arctan{x},y=arccot{x}\Rightarrow(x\in R,x\in(-\infty,+\infty))
  • y=\arcsin{x},y=\arccos{x}\Rightarrow(x\in[-1,1])

QickCmd

一、简介

  • 使用 Python 编写的自定义命令输入行，使用简单的代码实现较为基础的功能。
  • 可以同时使用系统命令不受影响，也可以通过自定义命令来对系统命令进行覆盖，也可以通过函数接收每次输入的命令和参数。

二、使用方法

  1. 基本使用

    from QickCmd import QickTerminal

cmd = QickTerminal('C4skg','./')

@cmd.startCmd()
def start(*args):
	print("命令返还 -> ", args);
